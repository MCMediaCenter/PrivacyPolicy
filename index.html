Personal information is not collected for the purpose of advertising or trading. No information which might be considered personal, or otherwise used to identify a user is collected by MC Media Center.

Third parties involved

The list of third parties might change over time, as features are added (or removed).
  • Microsoft provides the OS platform for running MC Media Center, store, reviews and email communications
  • Opensubtitles.org provides an optional service used to download subtitles for video files. This may be turned on by default. Some downloaded subtitle files may contain ads, provided by OpenSubtitles

Personal information gathered

All Mc Media Center apps WILL NEVER gather any personal information about its users. The other parties involved may collect user data. Consult their privacy policies for more information. Some services may require user accounts for accessing advanced features. Passwords are stored locally using standard Windows security features. Email addresses and other personal information may only be stored in order to provide extra functionality.

Support

mcosmin may respond to your reviews on the Windows Store from the email address mcmediacenter@outlook.com. I may require additional information on what you were doing when the application encountered a problem or what additional features you may want implemented in the app. Any personal information that could be extracted from email exchanges, including files, names, physical addresses will NOT be used by me in any way. Weather you want to contact me back is up to you.

Proximity & network

NFC

Applications can interact with NFC tags for automatization. This may happen silently, depending on OS settings

Network interactions

The application can detect other applications running on devices connected to the same wifi network via IP addresses. Users can opt out of network discovery from settings. I will not collect any information about the discovered devices. Devices running the application can send files between each other. Users can always opt out of file transfers completely or filter out files one by one. The application will store a local log of all files transfered and the remote end point. This log is for security purposes of the user and is NEVER uploaded to me in any way. Once the app is uninstalled, the log is gone. The application downloads Bing image of the day every day. This service is needed for the good function of some application features.

File access

The application needs access to your music, pictures and videos files in order to work properly. No information is sent over to third parties, except calculated hashes for video subtitle detection (there is no way to link a specific video file to a specific user). The application may save files to your disk in the following cases:
  • When editing metadata tags of music files: it will create a backup file and then save changes to your target file. The backup can be located in your music folder
  • When transferring files from other instances: The app will try to recreate the exact folder location of the file as on the other devices, but may fail to do so all the time. Transferred files will always be in your music or video folders
  • When creating cue sheets or m3u playlists.
  • When saving downloaded subtitles.

All other parties involved in this contract (such as Microsoft, Yahoo, Bing etc) have their own privacy policy. The application can interact with other applications through file picker contracts. Other apps may log or track what files you have been opening.
